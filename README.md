### How to Use This Code

1. **Download the files** from the `Indicators/` or `Strategies/` folders
2. **Open NinjaScript Editor** in NinjaTrader 8 (Tools ‚Üí NinjaScript Editor)
3. **Import or copy/paste** the code into a new indicator/strategy
4. **Compile** (F5)
5. **Add to your chart** and customize

Each file includes inline comments explaining what the code does.
